# Michigan Housing Forecasting

This project analyzes housing price trends in Michigan and builds a forecasting model to predict median housing prices over time.

## Objective
The goal is to understand housing market trends and create a simple predictive model using historical housing data.

## Tech Stack
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Project Structure

data/  
Raw and processed housing datasets

notebooks/  
Data cleaning, exploratory analysis, and modeling notebooks

src/  
Reusable scripts for preprocessing, feature engineering, training, and evaluation

outputs/  
Saved predictions and visualizations

## Model
A baseline Linear Regression model is used to forecast housing prices based on time features.

## Results
The model captures the upward trend in Michigan housing prices and produces predicted price estimates over time.

Predictions are saved in:
outputs/predictions/housing_predictions.csv
